Markit On Demand — Lookup API Sample

The Company Lookup API is useful for determining the stock ticker or symbol a company’s stock is traded with. For example, Google’s symbol is GOOG. Apple’s is AAPL. To use our APIs, a symbol is required as one of the inputs. The Company Lookup API response includes the company’s symbol, its full name, and the exchange it’s traded on.

Read the Lookup API documentation or check out our other data APIs.
